本文研究 Attention Residuals 在 SMAC 多智能体强化学习任务中的迁移适配性问题。
Attention Residuals 原本面向深层语言模型中的 residual stream 信息选择，而经典 PyMARL 中的 recurrent agent 通常较浅，直接复用该结构是否适合协同控制任务仍需实验验证。
本文在保持 QMIX、IQL、VDN 等算法主体不变的前提下，将 Attention Residuals 接入 RNN agent 的 hidden state 与 Q head 之间，并比较 Full、Lightweight、Block 以及 depth-only control 多种变体。
实验结果表明，重型 Full/Block AttnRes 直接迁移到浅层 MARL agent 后表现不稳定且训练成本较高；Lightweight AttnRes-L2 在完整 5M \code{5m\_vs\_6m} 对照中仅呈现较弱的 final-win 与 AUC 正向均值，且 seed 间方差较大。
进一步的 AttnComm-QMIX-L2 初步扩展实验显示，将 selective attention 转移到 agent-wise communication 维度后，在 \code{5m\_vs\_6m} 上相对 QMIX baseline 呈现更清晰的 final win 与 AUC 正向信号，其中 other-only communication 的提升更明显。
因此，本文将当前结果定位为结构迁移适配性实验研究，而不是新的强性能算法主张，并进一步指出 agent-wise communication 可能是比 depth-wise residual transfer 更符合 MARL 协同瓶颈的适配方向。
